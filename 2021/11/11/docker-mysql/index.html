配置远程访问

说明：

安全起见，原则上不允许 root 用户远程访问

MySQL 8有新的安全要求，不能像之前的版本那样一次性创建用户并授权需要先创建用户，再进行授权操作

进入容器登录 MySQL

查看数据库

创建用户

username：自定义用户名

host：登录域名，为 % 时表示任意 IP 可访问，即远程登录；为 localhost 时表示本机，或填写指定 IP 地址访问

password：自定义密码

用户授权

all：所有权限

第一个 * ：表示数据库，也可指定数据库名

第二个 * ：表示数据表，也可指定数据表名

with gran option：表示该用户可给其它用户赋予权限，但不可能超过该用户已有的权限
